# Portfolio

Hello, this is my personal portfolio.

Built using Vite and React.

Here you can see some of the projects I've worked on in my professional career and some personal projects.

## Infrastructure

The infrastructure for this project can be found at [Github Portfolio Infra](https://github.com/KristijanJ/portfolio-infra)

## Github actions

The code in this project is deployed using a manually triggered CI/CD pipeline using github actions. See `.github/workflows/deploy.yml`.

There is also a rollback option by specifying a Commit SHA to rollback to. See `.github/workflows/rollback.yml`.

The actions for this project might not be needed seeing as how I am the only developer working on it, but they are mostly used as a showcase of my ability to work with Github actions.
